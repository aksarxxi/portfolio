Projects

Health Advisor System

The Health Advisor System is a Python-based application that provides personalized health recommendations by calculating the Body Mass Index (BMI) from a user's height and weight. The system prompts the user to input their height and weight, calculates their BMI, and then offers advice based on the results. Depending on the BMI value, the system suggests appropriate actions, such as maintaining a healthy diet, increasing physical activity, or consulting a healthcare professional. The project is designed to be user-friendly, ensuring that health advice is both accessible and easy to understand.

Technologies Used: Python, BMI Calculation, Tkinter.

Animal Species Prediction

This web-based application, built using Django, predicts animal species based on user input or image uploads. It utilizes machine learning models trained on datasets from Kaggle to classify species such as tigers, pandas, and elephants. Users can either input characteristics or upload images, and the system returns predictions along with confidence levels, offering an interactive and educational experience.

Technologies Used: Django, Python, TensorFlow, Keras, Kaggle Datasets.
